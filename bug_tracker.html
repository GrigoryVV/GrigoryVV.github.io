Описание

TrackYourBugs-webApp - это небольшое web приложение для отслеживания багов в небольших проектах. Оно было создано с использованием "чистого" JavaScript и PHP в качестве языка backend. Проект создавался для использования его функционала в будущих собственных домашних проектах и получения необходимой практики работы с JavaScript.

Задачи, которые были решены при разработке проекта:

  • Отработать взаимодействие с элементами DOM посредством vanilla JavaScript
  • Освоить взаимодействие с сервером с помощью AJAX запросов
  • Создать инструмент валидации форм
  • Освоить обработку данных ответа сервера
  • Сделать интерфейс приложения адаптивным к экранам планшетов

В рамках данного проекта я использовал следующие инструменты и технологии:

  • Vanilla JavaScript
  • XMLHttpRequest
  • Gulp.js для минификации и сборки
  • Materialize CSS для стилизации
